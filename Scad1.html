Scada - Modbus protocol

Supervisory Control and Data Acquisition (SCADA) is a system that collects data from various digital and analogue inputs (sensors, switches etc) coming from factory, plant or other location, that sends data to a central computer which manages and controls data.
Main components of that system are: Human-Machine Interface (HMI) displays data to human operator, SCADA Master collects all data from different devices and controls the entire process, Remote Terminal Units (RTU) converts signals to digital data and sends to supervisory system, Programmable Logic Controller (PLC) field devices that controls input and output data from input devices.
Basic protocol used to connect devices to SCADA is Modbus (default port 502) which can be exploited even by using Metasploit which may be consider as low hanging fruit if attacker will prepare well planned attack.
