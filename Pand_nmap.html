Pandora's box -

Pandora’s box got new feature of Network scanner option, based on nmap scanning.

Network buster is script based device which can be plugged in to Ethernet slot or work on WiFi networks.
RJ45 interface device is plugged in to Raspberry Pi via USB (1,5 m cable) which can be easily hidden in sleeve, where different RJ-45 “endings” can be added (xross wired ending, usual switch ending)
