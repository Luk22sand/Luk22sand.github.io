Pandora's box

Pandora’s box is final project summarizing all devices in one place – platform. Project has been inspired by “Flipper ZERO” device, but in larger scale – literately.

Pandora is armed with:
1. WRT pi 4 router – broadcasts WiFi access point to control Pandora's components
2. Raspberry 3B+ - core unit managing Wifi attack, SDR read/write signals (check Pi box and SDR projects), contoling power supply to NodeMCU
3.NodeMCU - Wifi deauth, replication device (check NodeMcu project) controlled by separate power bank and relay to launch it only when it's needed
4.RTL-SDR usb - SDR dongle with 1Ghz antenna to capture signals

Next projects will add Arduino and Raspberry Pico devices (jammer, web server) to make platform more useful.
